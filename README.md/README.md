# IPL Analysis Dashboard

Analyzed 1090+ IPL matches and 260,000+ ball-by-ball deliveries to uncover team performance patterns, player stats, and toss impact.

## Key Insights
- Mumbai Indians are the most successful IPL team with 144 wins
- CSK is the most consistent team with 58.6% win rate
- Teams that choose to field first win 53.9% of the time vs 45.4% when batting
- V Kohli is the all-time top scorer with 8,014 runs
- YS Chahal leads wicket takers with 205 wickets
